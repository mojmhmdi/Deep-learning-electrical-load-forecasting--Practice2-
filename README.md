# paper with code

- This repository contanins the codes and data for electrical load forecasting model using LSTM. The cell by cell description is provided in the notebook.
- The codes are written in Python 3.10.
- The data is related to hourly load demand of Johor city in malaysia.
- The results are also presented in the section 6.2 of the following paper:
  https://www.researchgate.net/profile/Mojtaba-Mohammadi-11/publication/350237035_Stochastic_Energy_Management_and_Scheduling_of_Microgrids_in_Correlated_Environment_A_Deep_Learning-Oriented_approach/links/605e38ca92851cd8ce6c1697/Stochastic-Energy-Management-and-Scheduling-of-Microgrids-in-Correlated-Environment-A-Deep-Learning-Oriented-approach.pdf
